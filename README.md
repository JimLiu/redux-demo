# Redux Demo
Show how to use react + redux + react-router + immutable to build an app.


## SETUP

### Prerequisites for developing Kitematic
You will need to install: 
- [Node.js](https://nodejs.org/) v4.2+ 

### Getting Started

- `npm install`

To run the app in development:

- `npm start`
